1. Clone this repo, `git clone https://github.com/Codewebs/ks-mob.git/ `
2. Go to project's root directory, `cd <your project name>`
3. Run `Visual Studio` or Other IDE to Load dependencies
4. Build project and run IT
5. Enjoy!!!
